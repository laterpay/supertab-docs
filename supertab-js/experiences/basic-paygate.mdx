## `startPaygate`

```javascript
const supertabClient = new Supertab({ clientId: "test_client.abc" });

const supertabPaygate = supertabClient.startPaygate({
  experienceId: "experience.abc",
  onPurchaseCompleted: () => {},
  onPurchaseCanceled: () => {},
  onPriorEntitlement: () => {},
  onError: () => {},
});
```

### Parameters

<ExperiencesConfigParameters />

### Return value

A promise which resolves with an object with following properties:

<ResponseField name="isShowing" type="boolean">
  Whether paygate UI is currently visible.
</ResponseField>

<ResponseField name="toggle" type="function">
  Toggle visibility of paygate UI.
  <Info>
    If current user has prior entitlement to the content, the paygate will not
    show.
  </Info>
</ResponseField>

<ResponseField name="show" type="function">
  Show UI of the paygate.{" "}
  <Info>
    If current user has prior entitlement to the content, the paygate will not
    show.
  </Info>
</ResponseField>

<ResponseField name="hide" type="function">
  Hide UI of the paygate.
</ResponseField>

<ResponseField name="logIn" type="function">
  Start the login flow. Opens Supertab SSO popup window if user is unknown.
  Otherwise will log user in silently if auth data are found in the browser
  storage.
</ResponseField>

<ResponseField name="destroy" type="function">
  Destroy paygate instance. This removes all nodes related to paygate from DOM.
</ResponseField>
