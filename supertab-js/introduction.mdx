Supertab JS is a browser JavaScript library. It provides a layer handling authentication, purchases, and payments using Customer API.

Supertab JS can be used to [start experiences](/supertab-js/experiences/starting-experiences) on your pages and to interact with the [low-level SDK](/supertab-js/reference/overview) to consume API responses directly and build custom flows.

This reference guide documents installation, initialization, and all methods available in `Supertab` class.
